A origem da internet

Postado em 09/12/2022

A Internet é a maior rede mundial de computadores existente na atualidade, proporciona a seus usuários comunicação a baixo custo e acesso a fontes inesgotáveis de informação. Ela conecta pessoas para os mais variados fins e tem contribuído para ampliar e democratizar o acesso à informação, eliminando barreiras como distância, fronteiras, fuso-horário, etc.

Como funciona o HTML

Postado em 29/11/2022

A sigla HTML é o termo em inglês HyperText Markup Language seu código é utilizado uma página web. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. O HTML é uma linguagem de marcação de texto(e não de programação, como muitos às vezes confundem. Vale mencionar que, apesar de HTML ser usado para estruturar uma página, ele não é o responsável pela sua aparência.Para isso, existe o CSS (Folhas de Estilo em Cascata) permite a você criar páginas web agradáveis.

O que é CSS?

Postado em 09/11/2022

O termo em inglês CSS que, traduzido para o português, significa Folha de Estilo em Cascatas. É uma "folha de estilo" composta por camadas com Seletor ele seleciona o(s) elemento(s) a serem estilizados (nesse caso, elementos. A declaração uma regra simples como color: red; especificando quais das propriedades do elemento você quer estilizar e a propriedade a forma pela qual você estiliza um elementos HTML. Nesse caso, color é uma propriedade dos elementos em CSS, você escolhe quais propriedades você deseja afetar com sua regra e o valor da propriedade
