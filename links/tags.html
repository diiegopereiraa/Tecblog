O que é HTML?

O HTML é uma linguagem de marcação de texto seu código é utilizado uma página web. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens e tabelas. O HTML é uma linguagem de marcação (e não de programação, como muitos às vezes confundem. Vale mencionar que, apesar do HTML ser usado para estruturar uma página, ele não é o responsável pela sua aparência.Para isso, existe o CSS (Folhas de Estilo em Cascata) permite a você criar páginas web agradáveis.

O que são Tags?

As tags HTML são elementos que definem o conteúdo dos documentos HTML.Aplicar uma tag HTML é muito simples. Primeiro, você precisa escolher a tag que deseja aplicar. Em seguida, basta inserir o texto ou o conteúdo que deseja marcar com a tag. Uma tag HTML é uma marca que indica ao navegador que um determinado texto ou elemento deve ser interpretado conforme as regras do HTML.Elas são formadas por uma abertura e uma fechadura, delimitadas por um símbolo de menor que (<) e maior que (>), respectivamente. As tags podem conter atributos, definidos pelo símbolo de igual (=) e separados por espaços. Os atributos podem ter valores atribuídos a eles, que devem estar entre aspas duplas (“).

Tags semânticas: São tags que possuem um significado, ou seja, elas irão dar sentido a uma informação de texto ao navegador e buscador. As mais comuns são: article, header, nav e outras.

Tags sem semântica: São tags que não possuem significado. Como assim? Elas servem apenas para utilização de separação e estilização. As mais usadas são: b, i, span e outras.

!DOCTYPE HTML: Não é uma TAG é uma declaração de tipo documento uma maneira de informar o navegador o tipo de documento HTML.

html: Define o início do documento html, todos as demais tags devem estar dentro desta tag.

lang="pt-br": Informa a linguagem do tipo do documento.

meta charset="UTF-8": É indicação de um formato de codificação definido por caractéres.

head: Define o cabeçalho do documento com informações que não serão exibidas dentro do conteúdo da página.

title: Define o título do documento. é o texto mostrado na barra de título do navegador ou na aba da página.

body: Define o corpo do documento. O conteúdo de um documento html, como título, parágrafos, imagens,hiperlinks, tabelas, listas, etc.

div: É a tag que define uma divisão ou uma seção em um docuemnto html.

hgroup: Permite que o cabeçãho principal de uma seção de documento seja agrupado com quaiquer cabeçalhos. Secundário como subtitulos ou títulos alternativos para formatar um cabeçalho de vérios níveis.

header: É uma tag de cabeçalho que remete a semântica.

br: É a tag que significa quebra de linha ou break row.

nbsp: Non-breaking-space que significa espaço sem quebra.

p: Essa tag define um parágrafo do texto.

wbr: Essa tag Word break opportunity especifica onde um texto seria correto adicionar uma quebra de linha.

shy : Significa que adicionará um - se a palavra for muito longa, caso contrário, apenas renderizará a palavra como está.

figure: Essa tag serve para exibir imagens de qualquer tipo em nosso site de forma semântica.

figcaption: É a teg para adicionar uma descrição a imagem junto com a tag "figure".

href: Significa Hypertext reference, ou referência de hipertexto quando vou chamar um arquivo de fora.

h1 a h6: Representam 6 níveis de títulos, sendo o h1 de maior importância.

img: Define a utilização de uma imagem (JPG, GIF, PNG e outras). A imagem deve ser apontada com o uso do atributo src.

mark: representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.

small: Essa tag representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada.

i: Essa tag transforma o conteúdo em itálico;

u: Essa tag é usada para sublinhar o texto dentro da tag. Essa tag geralmente é usada para sublinhar palavras com erros ortográficos.

strong: Essa tag dá ao texto uma forte ênfase,e significa que o texto é exibido em negrito pelo navegador.

strong: Essa tag dá ao texto uma forte ênfase,e significa que o texto é exibido em negrito pelo navegador.

ol: "Ordered List Element" A tag é um elemento HTML que representa uma lista ordenada.

ul:"Unordered List Element" A tag é um elemento HTML que representa uma lista não ordenada.

li:"List Item Element" A tag define um item de lista.

a: Essa tag define um hiperlink, que é usado para vincular de uma página a outra.

hr:"Horizontal Rule" é usado para inserir uma linha horizontal numa página HTML.

sub:"Subscript Element" ou subscrito é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

sub:"Superscript Element" ou sobrescrito é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo.

blockquote: "Block Quotation" ou citação é utilizado para inserir um comentário de forma destacada com recuo de margem e espaçamento diante da linha anterior e diante da linha posterior.

font:"Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

del:"Font Element" ou letras é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML.

abbr: "Abbreviation Element" ou abreviatura é o marcador utilizado para se inserir uma sigla em uma página HTML, ele pode acontecer, por exemplo, dentro de um parágrafo.
